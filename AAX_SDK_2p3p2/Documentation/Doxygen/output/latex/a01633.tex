Registers one or more algorithm processing entrypoints (process procedures) 

Any non-\/overlapping set of processing entrypoints may be specified. Typically this can be used to specify both Native and TI entrypoints using the same call.

The A\+AX Library implementation of this method includes backwards compatibility logic to complete the Process\+Proc registration on hosts which do not support this method. Therefore plug-\/in code may use this single registration routine instead of separate calls to \mbox{\hyperlink{a01625_a4acfd619cfbd35ffd0ff20ca67bda305}{Add\+Process\+Proc\+\_\+\+Native()}}, \mbox{\hyperlink{a01625_ae1d1a2fe050631ab50720d46358e9605}{Add\+Process\+Proc\+\_\+\+T\+I()}}, etc. regardless of the host version.
