\hypertarget{a00815}{}\section{Taper delegates}
\label{a00815}\index{Taper delegates@{Taper delegates}}


\subsection{ }
Classes for conversion to and from normalized parameter values. 

Taper delegates are used to convert real parameter values to and from their normalized representations. All taper delegates implement the A\+A\+X\+\_\+\+I\+Taper\+Delegate$<$\+T$>$ interface template, which contains two conversion functions\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{virtual} T       NormalizedToReal(\textcolor{keywordtype}{double} normalizedValue) \textcolor{keyword}{const} = 0;}
\DoxyCodeLine{\textcolor{keyword}{virtual} \textcolor{keywordtype}{double}  RealToNormalized(T realValue) \textcolor{keyword}{const} = 0;    }
\end{DoxyCode}


In addition, tapers may incorporate logical value constraints via the following interface methods\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{virtual} T       GetMaximumValue()  \textcolor{keyword}{const} = 0;}
\DoxyCodeLine{\textcolor{keyword}{virtual} T       GetMinimumValue() \textcolor{keyword}{const} = 0;}
\DoxyCodeLine{\textcolor{keyword}{virtual} T       ConstrainRealValue(T value) \textcolor{keyword}{const} = 0;}
\end{DoxyCode}
